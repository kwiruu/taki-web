## Set theme in config

```json
{
  "ui": {
    "theme": "vscode-dark-plus"
  }
}
```

## Choosing a theme

<Card className="my-6">
  <CardHeader>
    <CardTitle>Recommendation</CardTitle>
    <CardDescription>
      Pick based on contrast and session length, not aesthetics alone.
    </CardDescription>
  </CardHeader>
  <CardContent>
    For long debugging sessions, use high-contrast dark themes such as
    `vscode-dark-plus` or `terminal-nord`. For bright rooms, `vscode-light-plus`
    or `default-white` may reduce eye strain.
  </CardContent>

## Fast switch workflow

1. Edit `taki.json` and change `ui.theme`.
2. Save file.
3. Restart dashboard using `r` for focused service or restart full session.
