## Keyboard controls

| Key                             | Action                         |
| ------------------------------- | ------------------------------ |
| `q` / `Ctrl+C`                  | Quit and trigger shutdown flow |
| `o`                             | Open options panel             |
| `?`                             | Open shortcuts panel           |
| `r`                             | Restart focused service        |
| `1` / `2` / `3` / `4`           | Switch layouts quickly         |
| `Tab`                           | Move focus to next pane        |
| `Arrow keys` or `h` `j` `k` `l` | Move selection/focus           |

## Operational tips

1. Start with a grid or vertical layout when booting multiple services.
2. Switch to full-log when diagnosing one noisy service.
3. Keep `maxLogLines` high enough for debugging sessions.

## Typical session

```text
1) taki run
2) wait for services to become healthy
3) Tab to select a failing service
4) press r to restart
5) press ? when you need a shortcut reminder
```
