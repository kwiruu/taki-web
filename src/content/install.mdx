## Verify the binary

```bash
taki version
```

If the command is not found, restart the terminal and confirm your package manager global bin path is in `PATH`.

<Card className="my-6">
  <CardHeader>
    <CardTitle>Permission-safe install tip</CardTitle>
    <CardDescription>
      Avoid installing globals with elevated system permissions whenever
      possible.
    </CardDescription>
  </CardHeader>
  <CardContent>
    Use a user-scoped Node setup (for example via nvm, fnm, or Volta) so global
    packages are installed without `sudo`/administrator flows.
  </CardContent>
</Card>

## Initialize config

```bash
taki init
```

You can force regeneration if needed:

```bash
taki init --force
```

## First run

```bash
taki run
```

Equivalent shortcut:

```bash
taki
```

## Update and uninstall

```bash
# update
npm install -g @kwiruu/taki-cli@latest

# uninstall
npm uninstall -g @kwiruu/taki-cli
```
